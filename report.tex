    \subsubsection{2. Examine the State and Action
Spaces}\label{examine-the-state-and-action-spaces}

The simulation contains a single agent that navigates a large
environment. At each time step, it has four actions at its disposal: -
\texttt{0} - walk forward - \texttt{1} - walk backward - \texttt{2} -
turn left - \texttt{3} - turn right

The state space has \texttt{37} dimensions and contains the agent's
velocity, along with ray-based perception of objects around agent's
forward direction. A reward of \texttt{+1} is provided for collecting a
yellow banana, and a reward of \texttt{-1} is provided for collecting a
blue banana.
